## Project

A Virtual Tabletop platform, implemented as a physics sandbox, playable in the browser.
The repo will be hosted on Github, and deployed to AWS.

Built using ThreeJS, Typescript, React, and Websockets/WebRTC.
Provides a multiplayer game environment, with a chatroom, and a lobby for players to find each other.
Users can also use a scripting language to create and automate their own games, similar to Tabletop Simulator.
